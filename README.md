## 주요 기능
1. **회원 관리**
   - 카카오 소셜 로그인을 통한 간편한 회원가입 및 로그인
   - JWT 토큰 기반의 보안 강화된 인증 시스템

2. **맞춤형 영양 추천**
   - 사용자의 신체 정보를 바탕으로 일일 권장 영양소 계산 및 추천

3. **AI 기반 음식 인식**
   - 사진 한 장으로 음식 자동 인식 및 영양 정보 제공
   - 빠르고 정확한 식단 기록 지원

4. **식단 기록 및 분석**
   - 일일 섭취량 자동 계산 및 기록
   - 직관적인 UI를 통한 쉬운 식단 관리 및 분석
